formSubmit

  • function
 

Called when the form is submitted. Serializes the form data and compares it to the formObject. If there are changes, the object is updated and its save method is called. The event submit is dispatched.

static.formSubmit(scope, form, event)

Called when the form is submitted. Serializes the form data and compares it to the formObject. If there are changes, the object is updated and its save method is called. The event submit is dispatched.

Parameters

  1. scope {can.Map}

    The stache scope

  2. form {HTMLFormElement}

    The submitted form object

  3. event {can.Event}

    The submit event

Returns

{Boolean}

Returns false to prevent the form from actually submitting
