print

  • function
 

Generates a printout and returns a promise which resolves to an object with a url and title property.

function(options)

Parameters

  1. options {Object}

    The print out options.

    var options = {
    map: map,
    layout: '8.5x11 Landscape',
    title: 'My Map Title',
    dpi: 300
}
provider.print(options);

Returns

{Promise}

The promise that resolves to an object with a url and title property. If the print fails, the url will be null, and instead an object with a title and error property will be returned
